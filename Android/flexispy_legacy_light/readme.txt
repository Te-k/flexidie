Before build a release please check the following items:
    1. In com.vvt.android.syncmanager.Customization, turn off debug flag, check product edition, name, and ID.
    2. In com.fx.android.common.Customization, turn off the debug flag. 
    2. In /res/values/string.xml, check app name, build date, and build version.
    3. In AndroidManifest.xml, check app name and versions.
